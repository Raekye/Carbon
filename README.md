Carbon
======

Implementations of data structures and algorithms in various languages for educational purposes.

### Setup/Testing
##### Ruby
- `rake test`

##### Haskell
- `cabal configure --enable-tests && cabal build && cabal test`

##### Erlang
- `rebar compile eunit`
